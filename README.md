# Foodbridge — Purchase Settlement (discovery preview)

A **disposable discovery prototype** for feedback — static HTML/CSS/vanilla JS, no build, no server.
It mirrors the settlement reference UI to the supplier/purchase side: settlement list → inline
Expected·Actual·Difference expand → **View & Resolve** drawer (Details / Evidence / Activity Log /
Comments) → reason + impact preview → Resolve, plus **Bulk Resolve**.

**▶ Live:** see the repo's GitHub Pages URL (Settings → Pages), or open `screens/screen-01.html`.

Data is fake role-play seed (`seed-data/seed.js`). This is a **preview snapshot** — the source of
truth lives in the private module repo. Feedback welcome via Issues.
